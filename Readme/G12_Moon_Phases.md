## G12 Moon phases
 - simple ambient feature that adds moon phases to the game. (allows you to update moon textures)

    [![G1&2 Moon phases](https://img.youtube.com/vi/zc3lch6syaY/0.jpg)](https://www.youtube.com/watch?v=zc3lch6syaY)
